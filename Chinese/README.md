## JS代码示例：

```javascript
~~3.14
```
问题：上面的语句的返回值是什么？
**答案：3**

```javascript
"i'm a lasagna hog".split("").reverse().join("");
```
问题：上面的语句的返回值是什么？
**答案："goh angasal a m'i"** 

```javascript
( window.foo || ( window.foo = "bar" ) );
```
问题：window.foo的值是什么？
**答案："bar"**
只有window.foo为假时的才是上面答案，否则就是它本身的值。

```javascript
var foo = "Hello"; (function() { var bar = " World"; alert(foo + bar); })(); alert(foo + bar);
```
问题：上面两个alert的结果是什么
**答案: "Hello World" & ReferenceError: bar is not defined** 

```javascript
var foo = [];
foo.push(1);
foo.push(2);
```
问题：foo.length的值是什么？
**答案：'2'**

```javascript
var foo = {};
foo.bar = 'hello';
```
问题：foo.length的值是什么？
**答案: `undefined`

```javascript
foo = foo||bar
```
问题: 这行代码是什么意思? 
**答案: if(!foo) foo = bar

```javascript
foo>>1
```
问题: 这行代码是什么意思? 
**答案: Math.floor(foo/2)

```javascript
foo|0
foo+.5|0
```
问题: 这行代码是什么意思? 
**答案: parseInt(foo) & Math.round(foo)

```javascript
function foo(bar1, bar2, bar3){}
```
问题: 如何获取参数的个数?
**答案: foo.length //this example is 3
